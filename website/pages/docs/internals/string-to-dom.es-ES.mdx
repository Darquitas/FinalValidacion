# `stringToDOM(){:jsx}`

**Sintaxis:** `stringToDOM(content, svg){:jsx}`\
**Ejemplo:** `stringToDOM('<div>Hola Mundo</div>'){:jsx}`


La función `stringToDOM` toma una cadena de HTML o SVG y devuelve un objeto del DOM. Esto es útil para crear elementos del DOM a partir de cadenas de texto (string).

```jsx
import { stringToDOM } from 'million';

const dom = stringToDOM('<div>Hola Mundo</div>');

dom.innerHTML; // 'Hola Mundo'
dom.tagName; // 'DIV'

const svg = stringToDOM('<g><rect /></g>'); // crea SVG!
```
